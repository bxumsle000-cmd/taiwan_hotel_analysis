# 台灣旅館業市場分析

基於交通部觀光署 2017–2025 年旅館營運統計數據，評估台灣旅館業的進場可行性，並篩選具有潛力的區域。

## 分析摘要

本專案將分析分為兩個層次：

**第一層：市場基本面判斷**  
透過住房率、房間供給數量、平均房價等指標，判斷台灣旅館業整體的供需狀態。結論為三個指標同步正向——住房率回到疫前水準、供給增速趨緩、房價穩步上漲。

**第二層：區域篩選**  
將指標與 22 個縣市，透過「住用數成長率 vs 房間數成長率」及「住用率 vs 房間數成長率」兩張散佈圖，篩選供需缺口最明顯的區域。最終識別新竹市為一般旅館市場中具有潛力的區域——住用數成長率全台最高、房間數近乎零成長、住用率僅次於台北市。


## 資料來源

| 資料 | 來源 | 用途 |
|------|------|------|
| 一般旅館營運月報 | 交通部觀光署 | 各縣市一般旅館住房率、住用數、房價、營收 |
| 觀光旅館營運月報 | 交通部觀光署 | 各縣市觀光旅館住房率、住用數、房價、營收 |


## 使用工具

- **Python**：pandas、openpyxl，用於多年份月報的讀取、清洗與整合，並將處理後的資料導入 SQL 資料庫（`1_administration.ipynb`、`1_general.ipynb`、`2_load_to_db.ipynb`）

- **MySQL**：Create View，建立分析用的彙總檢視表（`sql_scripts/2017_to_2025旅館營運月報.sql`）

- **Power BI**：互動式 Dashboard，包含趨勢圖、散佈圖、縣市篩選器與預估營收計算（`旅館.pbix`）


## 使用方式

1. 確認 `一般旅館_all.xlsx` 和 `觀光旅館_all.xlsx` 已下載至專案資料夾（原始資料可從[觀光署行政資訊網](https://admin.taiwan.net.tw)下載）
2. 執行 `2_load_to_db.ipynb`，依提示輸入 MySQL 帳密，自動完成資料匯入與 View 建立
3. 開啟 `旅館.pbix` 連接資料庫即可使用 Dashboard
